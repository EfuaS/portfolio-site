About the Project

The Neighbor Care E-commerce Platform is my first try at building an ecommerce platform and the project I am most proud of yet. This project is the result of a contract with my Aunt to digitize her African print fabric shop to widen customer reach. Often times, the physical shop is closed due to her 9 - 5 work schedule on weekdays, thus this project is to provide a 24/7 open online store that ensures that customers could access the shop whenever needed.


The preliminaries of this project included an analysis of the business model and stakeholder engagement to develope an efficient system that fulfills business objectives. Visit Storefront, to learn more about my record digitization project.

About Neighbor Care

Neighbor Care is a registered African Fabrics and Cosmetics retail business in Ghana. Main products for sale include African print fabrics, personal care cosmetics from brands such as dove, nivea as well as high quality perfumes including brands like Channel and Dior.

Model-View-Controller Architecture


I also employed the Model View Controller Structure to reduce code redundancy, and maintain consistency and a more organized file structure. All communications with the database where handled in PHP classes and had their methods called from the controllers as shown below.


fig.3 sample modeles php file


fig.4 sample controllers php file



Similarly, to reduce code repetition, main sections of the page remained same across the platform such as the navigation bar , header, footer and page scripts were saved in a separate PHP file and then required in each page. This was effective in ensuring consistency across the platform since updated code in any of these files automatically reflected in all linked pages.



The image below shows how I separated the platform's footer, header, naigation bar and script files and saved them under a folder called includes on the left side of the image. Now, close to the top we can see how I called this page section by requiring them.
